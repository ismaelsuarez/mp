## CONFIG – Resultados

### Objetivo
Explorar, filtrar y exportar los datos del último reporte generado.

### Filtros
- Desde/Hasta
- Estado
- Buscar
- Generar (rango) / Reset

### Tabla y sumarios
- Paginación 10/20/50/100
- Totales de ingresos, devoluciones y neto

### Exportaciones
- CSV, XLSX, DBF, JSON
- Enviar por email (si SMTP configurado)

### Referencias
- `src/renderer.ts`, `src/main.ts`
